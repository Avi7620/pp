Berkeley Madonna - Mathematical Modeling Software

Berkeley Madonna is a mathematical modeling software package. It numerically solves ordinary differential equations and difference equations, originally developed to execute STELLA (short for Systems Thinking, Experiential Learning Laboratory with Animation) programs. Berkeley Madonna provides the facility of putting parameters into a slider that can in turn be moved by a user to change the value. Such visualizations enable quick assessments of whether or not a particular model class is suitable to describe the data to be analyzed and modeled, and, later, communicating models easily to other disciplines such as medical decision makers.
