REDCap - Research Data Capture

REDCap is a web-based application designed to help researchers collect, manage, and analyze data for their studies. It provides a user-friendly interface for creating electronic data capture forms, allowing researchers to collect data securely online. REDCap also offers features for managing project workflows, tracking data quality, and exporting data for analysis.

Installation Requirements

  • WAMP (Windows Apache MySQL PHP): Download WAMP
  • MySQL Database: Download MySQL
  • phpMyAdmin: Download phpMyAdmin
