Open Babel - Chemical Toolbox

Open Babel is an open-source chemical toolbox designed to interconvert between various chemical file formats. It provides a wide range of functionalities including file format conversion, chemical structure manipulation, 2D and 3D structure visualization, substructure searching, descriptor calculation, conformer generation, and chemical reaction conversion.
